  - 1️⃣ What is the difference between var, let, and const?
    = var is function scoped and it hoists on the other hand let and const both are block scoped. we use let to update the value and the const cannot be reassigned.

  - 2️⃣ What is the spread operator (...)?
    = spread operator is used to copy an array or object, also it expands an array or object into individual elements.

  - 3️⃣ What is the difference between map(), filter(), and forEach()?
    = map is used when we need to modify an array and it returns a new array. filter is also similar to map but we can set conditions into it and what ever element in the array passes through the condition will come out as new array. forEach is used when we don't need any return from it but we need to loop through for an array.

  - 4️⃣ What is an arrow function?
    = it is a shorter way to write a function and it was introduced in ES6.

  - 5️⃣ What are template literals?
    = template literals are backticks. we use it when we need multi line strings or we need to use any expression with ${}.
